## Prerequisites
Before diving into the Llama2 Medical Bot, ensure your system is equipped with:
- Python 3.6 or higher
- Necessary Python packages (installable via pip):
  - `langchain`
  - `chainlit`
  - `sentence-transformers`
  - `faiss`
  - `PyPDF2` (for PDF document loading)
